# ComfyUI Mini Nodes

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![ComfyUI](https://img.shields.io/badge/ComfyUI-0.3.0%2B-green)](https://github.com/comfyanonymous/ComfyUI)

> Simple custom nodes for ComfyUI: color matching, latent size, image saving.

## 📦 Features

- ✅ **Color Match Node** – Match colors between images (Based on the area of your costum input mask)
- ✅ **Latent Size Adjuster** – Choose latent size with presets (fast & efficient)
- ✅ **Image Save with Metadata** – Save images with your workflow.

## 🚀 Installation

1. Open your ComfyUI directory
2. Navigate to `custom_nodes/`
3. Run:
   ```bash
   git clone https://github.com/catmaxj/comfyui-mini-nodes.git
4. No dependencies requirements, just restart your ComfyUI and type the nodes name, drag them into your workflow.
